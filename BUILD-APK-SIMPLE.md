# 📱 ساخت APK - روش خیلی ساده (بدون کد!)

## روش 1: استفاده از PhoneGap Build (رایگان) ⭐

### مرحله 1: ثبت‌نام
1. به سایت برید: https://build.phonegap.com/
2. با GitHub login کنید (همون اکانتی که پروژه رو داره)

### مرحله 2: ساخت اپ جدید
1. روی **"+ New App"** کلیک کنید
2. ریپازیتوری `alarm_on_the_go` رو انتخاب کنید
3. روی **"Ready to build"** کلیک کنید

### مرحله 3: دانلود APK
1. منتظر بمونید تا build تمام شه (2-3 دقیقه)
2. روی **"Android"** کلیک کنید
3. فایل APK رو دانلود کنید
4. **تمام!**

## روش 2: استفاده از Appsgeyser (خیلی ساده!)

### قدم به قدم:
1. به سایت برید: https://appsgeyser.com/
2. روی **"Create App"** کلیک کنید
3. گزینه **"Website"** رو انتخاب کنید
4. URL اپتون رو وارد کنید:
   ```
   https://mehrabihojjat.github.io/alarm_on_the_go/
   ```
5. اطلاعات اپ رو پر کنید:
   - **App Name**: آلارم در حرکت
   - **Description**: آلارم مبتنی بر موقعیت
6. آیکون رو آپلود کنید (icon-512.png)
7. روی **"Create"** کلیک کنید
8. APK رو دانلود کنید

**مدت زمان**: فقط 5 دقیقه! ✨

## روش 3: استفاده از AppsBuilder

1. به سایت برید: https://www.apps-builder.com/
2. ثبت‌نام کنید (رایگان)
3. **"Create New App"** → **"WebView App"**
4. URL: `https://mehrabihojjat.github.io/alarm_on_the_go/`
5. تنظیمات:
   - Enable Location
   - Enable Notifications
   - Fullscreen Mode
6. Build & Download

## روش 4: استفاده از Convertify (سریع‌ترین!)

1. به سایت برید: https://convertify.io/
2. URL رو وارد کنید
3. تنظیمات:
   - Platform: Android
   - App Type: Progressive Web App
4. Generate APK
5. دانلود!

## بعد از ساخت APK:

### تست روی گوشی:
1. APK رو به گوشی انتقال بدید
2. نصب کنید
3. تست کنید که همه چیز کار می‌کنه

### برای Google Play Store:

اگه می‌خواید در Google Play منتشر کنید:

1. **اکانت Developer بسازید**: https://play.google.com/console/signup
   - هزینه یکبار: $25

2. **App Bundle بسازید** (بجای APK)
   - در سرویس‌های بالا، گزینه "AAB" رو انتخاب کنید

3. **فرم‌های مورد نیاز رو پر کنید**:
   - توضیحات اپ
   - اسکرین‌شات‌ها
   - Privacy Policy

4. **Submit کنید**
   - بررسی Google: 2-7 روز
